## What I do

I build NLP and LLM systems, and I ship open-source projects every week.

By day I work at **AI Buddy Catalyst Labs** building LLM-integrated pipelines, AI agent systems, and workflow automation. On the side I publish one NLP/AI project a week -- ranging from Bangla text preprocessing to LLM evaluation suites, RAG pipelines, and conversational AI.

My research focus is **NLP for low-resource languages**, particularly Bangla, where standard tools silently fail due to Unicode edge cases most engineers never think about. I care about making NLP accessible for the 230 million people who speak Bangla.

## Featured project

### [bangla-text-toolkit](https://github.com/Mouly22/bangla-text-toolkit) &nbsp; [![PyPI version](https://badge.fury.io/py/bangla-text-toolkit.svg)](https://badge.fury.io/py/bangla-text-toolkit) [![CI](https://github.com/Mouly22/bangla-text-toolkit/actions/workflows/ci.yml/badge.svg)](https://github.com/Mouly22/bangla-text-toolkit/actions) ![Python](https://img.shields.io/pypi/pyversions/bangla-text-toolkit) ![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)

Zero-dependency Python library for Bangla NLP preprocessing. Handles the Unicode problems that silently break standard NLP tools on Bangla text.

```python
from bangla_text_toolkit import Pipeline

pipe = Pipeline(remove_english=True, remove_digits=True)
pipe.process("আমি  বাংলা  ভালোবাসি!!! https://example.com 😊")
# 'বাংলা ভালোবাসি'
```

**Why it exists:** Python's `\w` doesn't match Bangla combining characters (vowel signs / matras). Most preprocessing tools silently corrupt Bangla text. This one doesn't. 51 tests. Zero dependencies. Supports Python 3.9+.
