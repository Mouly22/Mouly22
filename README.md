## About

I'm a software engineer specializing in NLP and AI systems. By day I build LLM-integrated pipelines, AI agents, and workflow automation at AI Buddy Catalyst Labs. On the side I ship one NLP/AI project a week — RAG pipelines, LLM evaluation frameworks, conversational AI, and text processing tools.

I care about building things that actually work in production, across languages and edge cases that most engineers skip.
