## Featured project

### [bangla-text-toolkit](https://github.com/Mouly22/bangla-text-toolkit) &nbsp; [![PyPI version](https://badge.fury.io/py/bangla-text-toolkit.svg)](https://badge.fury.io/py/bangla-text-toolkit) [![CI](https://github.com/Mouly22/bangla-text-toolkit/actions/workflows/ci.yml/badge.svg)](https://github.com/Mouly22/bangla-text-toolkit/actions) ![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)

Production-grade Bangla NLP preprocessing in Python. Unicode normalization, tokenization, text cleaning, stopword filtering. Zero dependencies.

```python
from bangla_text_toolkit import BanglaTextPipeline

pipe = BanglaTextPipeline.default()
pipe.run("\u0986\u09ae\u09bf \u09ac\u09be\u0982\u09b2\u09be \u09ad\u09be\u09b2\u09cb\u09ac\u09be\u09b8\u09bf!!! https://example.com")
# \u2192 '\u0986\u09ae\u09bf \u09ac\u09be\u0982\u09b2\u09be \u09ad\u09be\u09b2\u09cb\u09ac\u09be\u09b8\u09bf'
```

Built because Python's `\w` doesn't match Bangla combining characters — most NLP tools silently corrupt Bangla text. This one doesn't.
